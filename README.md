# tslint-wix-react
A tslint configuration extending tslint-microsoft-contrib in a specific to wix ways

## How to use
In your project:
```
$ npm i --save-dev tslint-wix-react
```
In your `tslint.json`:
```
extends: ["tslint-wix-react"]
```
